FEATURES
Beneficiate of an efficient drag and drop experience to create your stacks

One of our primary objectives is to ensure your global experience being amazing. Therefore, we put a lot of effort on the usability of the user interface, allowing you to drag and drop elements in a very natural and fluid way to create your cloud computing stacks.

The objective has been reached: you can almost not see the difference between MadeiraCloud Web IDE and a native application.

Validate and estimate the cost of your stacks directly from the IDE

Your time is precious, and we know this! Thus, MadeiraCloud Web IDE now integrates Trusted Advisor in order to validate the stacks you want to launch, giving you some advices to make it better and listing the actual errors. No more time wasted here!

Simply provision your stacks to your Amazon Web Services account

Once your stacks are designed, you can decide to run them on AWS using MadeiraCloud IDE, transforming them as apps. Nothing to do there, we provision everything in one click!

Your ideas grow, your apps too! Freely edit the running apps

In a world where everything changes and evolve quickly, we let you update your running apps as you feel like it directly from MadeiraCloud IDE.

Find yourself on a familiar ground with most common AWS resources supported

MadeiraCloud IDE now supports the most common AWS resources (EC2, VPC, Auto Scaling, Elastic Load Balancing, EIP ...), including advances features for all of them.

In constant evolution, we can participate to the debate, voting for the next released features on our Trello public board.

Manage your Security Groups easier than ever

The Security Groups acts like a security layer on AWS. MadeiraCloud IDE helps you to define the set of security rules that will generate the virtual links between your resources.

No more worries, it has now become as simple as a click!

Group your instances for more productivity

Aware that your primary concern with visual tools can repeatability, MadeiraCloud IDE allows you to create some groups of instances, avoiding the recurrent and unproductive tasks.

New! - Automate your software configuration (Invite Only)

But MadeiraCloud IDE is more than a simple graphical interface. Today, we let you configure and manage the software configuration of your instances in a fully automated way!

Get more details about VisualOps

Be free; export the stacks you've designed

MadeiraCloud IDE let you export the stacks that you have designed into image files, CloudFormation scripts, or even our json format that you can import later in the same, or another account.

New! - Visualize your unmanaged VPCs! (Invite Only)

You have created a VPC earlier using the AWS console or CloudFormation? Its architecture is growing and you would like to use the power of MadeiraCloud GUI to visualize it? It is now possible using the new VPC visualization feature!

Global dashboard

MadeiraCloud IDE provides you a screen that delivers you all the information that you actually need. Thus, we gathered in the new dashboard all the important details concerning your existing stacks and applications.

Furthermore, we added the most important details about your AWS resources from your AWS account, giving you a perfect overview of what you are currently using.
